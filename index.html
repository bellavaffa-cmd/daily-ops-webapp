Scan Order from WMS

Skip retyping order details — reads the order open on a wms.golocad.com Packing Station and opens the Report an Issue form with Order ID, Tote, Brand, Warehouse, and AWB Error Code already filled in. Two options below — the extension is recommended.

RECOMMENDED Chrome Extension

Auto-shows a Report Error button whenever an order is loaded. When a carrier/AWB error appears, it automatically opens the form before the error toast disappears — no manual click needed.

  1. Download the extension zip:
    Download Extension
  2. Unzip the downloaded file.
  3. Open chrome://extensions, enable Developer mode (top-right toggle), then click Load unpacked and select the wms-extension folder.
  4. Done. Open any order on the WMS Packing Station — a red Report Error button appears in the bottom-right corner.
or use the bookmarklet
Bookmarklet

No install needed — drag the button below to your bookmarks bar, then click it while an order is open on the WMS screen. You must click it manually before the error toast disappears.

  1. Install. Drag to your bookmarks bar (⌘⇧B / Ctrl+Shift+B to show it):
    📋 Scan WMS → Report Error
  2. Use it. Open an order on the WMS screen, then click the bookmark. For carrier errors, click immediately after "Complete Packing And Create Label" — the error toast dismisses within a few seconds.
